# VTracker Video Event Stream (TypeScript)
VTracker is tracking VTuber overworld using event stream.

## Setup
```
    // Install library
    yarn install 

    // Config file in config/config.example.json
    // Change name config.example.json to config.json
    // Then run 
    yarn start
```
